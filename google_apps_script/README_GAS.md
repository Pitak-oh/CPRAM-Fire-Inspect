# 📱 แผนการติดตั้งระบบตรวจสอบถังดับเพลิงอัจฉริยะ ซีพีแรม (สุราษฎร์ธานี) เป็น Web Application ผ่าน Google Apps Script

เพื่อรองรับการใช้งานบนระบบ **iOS (iPhone/iPad)** และ **Android** ได้อย่างสะดวกสูงสุด โดยไม่ต้องทำเรื่องติดตั้งผ่าน App Store/Play Store ให้ยุ่งยาก ท่านสามารถตั้งค่าระบบ Web Application บนเซิร์ฟเวอร์ระบบคลาวด์ของตนเองผ่าน **Google Apps Script** ที่เชื่อมกับ **Google Sheets** (ฐานข้อมูล) และระบบแจ้งเตือนทางไลน์กรุ๊ป (**LINE Notify**) ดังนี้ครับ:

---

## 🌟 ฟังก์ชันที่มีให้ใช้งานในโปรเจกต์นี้
1. **ระบบลงชื่อเข้าแบบพนักงานและผู้จัดการ (Dual-Role Login)**: แยกระหว่าง "พนักงานตรวจ (User)" และ "ผู้จัดการ (Admin)"
2. **ระบบความปลอดภัยสูงสุด (2FA OTP Verification)**: มีการตรวจสอบชั้นที่สอง ป้องกันการบันทึกข้อมูลสวมสิทธิ์
3. **ฟังก์ชันสแกน QR Code หน้างานจริงด้วยกล้องมือถือ**: รองรับการสแกนสดจากกล้อง iOS/Android ผ่านโปรแกรมโดยไม่ต้องสลับแอป
4. **แบบประเมินความปลอดภัยดัชนี ISO 5 ประเด็นหลัก**: ตรวจสอบเกจวัดแรงดัน, สลักล็อกร้อยซีล, สายฉีดฉีดดับไฟ, ผิวตัวถัง และถ่ายถาพสถานะและบันทึกข้อคิดเห็น
5. **ถ่ายภาพยืนยันตนและเก็บหลักฐานลง Google Drive**: บันทึกรูปภาพโดยตรงลงใน Drive อย่างปลอดภัย
6. **ระบบซิงก์ประวัติออฟไลน์ (Offline Mode)**: หากพนักงานตรวจสอบในที่อับสัญญาณคลังเยือกแข็ง ระบบจะบันทึกเก็บที่เครื่องไว้ก่อน และทำการซิงก์ส่งข้อมูลอัตโนมัติเมื่อตรวจจับเน็ตได้
7. **แดชบอร์ดความปลอดภัยของผู้จัดการแบบเรียลไทม์**: แสดงผลสัดส่วนเสถียร สรุปสำเร็จพิกัด และแสดงพิกัดถังดับเพลิงผ่านแผนภาพ (Factory Floor Plan) แบบมีปฏิสัมพันธ์ (Interactive Map)
8. **ระบบสั่งพิมพ์ QR Code ติดตัวถังจำลอง**: สั่งพิมพ์พิกัดพร้อมนำไปใช้งานหน้างานได้ทันที
9. **ระบบแจ้งเตือนเหตุขัดข้องผ่านไลน์กลุ่มผู้จัดการทันที (LINE Notify Alarms)**: หากตรวจพบว่าถังมีอาการชำรุด เช่น แรงดันบกพร่อง หรือสายฉีดอุดตัน ระบบจะส่งสัญญานเตือนแจ้งผู้ดูแลให้พร้อมเข้าหน้างานภายใน 5 วินาที
10. **ระบบส่งออกสถิติ PDF รายงาน และ Excel**: ได้รับความสะดวกในการส่งเอกสารประเมินผล

## 🛠️ ขั้นตอนจัดตั้งระบบ (ภายในเวลาจำกัด 5 นาที)

### ขั้นตอนที่ 1: เตรียมฐานข้อมูลบน Google Sheets
1. เปิดเว็บไซต์ [Google Sheets](https://sheets.google.com) และกดสร้าง **สเปรดชีตเปล่า** ขึ้นมาใหม่ 1 ไฟล์
2. ตั้งชื่อไฟล์สเปรดชีต เช่น `CPRAM_Fire_Dashboard_DB` ให้ตรงความต้องการ

### ขั้นตอนที่ 2: วางซอร์สโค้ดใน Apps Script
1. ที่แถบเมนูด้านบนของสเปรดชีต ให้ไปที่เมนู **ส่วนขยาย (Extensions)** -> **Apps Script**
2. ลบโค้ดทั้งหมดที่แสดงอยู่ในไฟล์เริ่มต้น `Code.gs` ออก
3. เปิดคัดลอกรหัสต้นฉบับจากไฟล์ `/google_apps_script/Code.gs` ของโครงการนี้ทั้งหมด แล้ววางทดแทนลงไป
4. สร้างไฟล์ HTML เพิ่มเติมจำนวน 2 ไฟล์ใน Apps Script ดังนี้:
   * **ไฟล์แรก**: กดที่ปุ่มกลมบวก (`+`) มุมมองรายชื่อไฟล์ในหน้าจอด้านล่าง -> เลือก **HTML** ตั้งชื่อว่า `Index` และนำโค้ดในไฟล์ `/google_apps_script/Index.html` ไปใส่ทดแทน
   * **ไฟล์ที่สอง**: กดปุ่มบวก (`+`) เลือก **HTML** ตั้งชื่อว่า `Dashboard` และนำโค้ดในไฟล์ `/google_apps_script/Dashboard.html` ไปใส่ทดแทน
5. กดปุ่ม **บันทึกโครงการ** (รูปแผ่นดิสก์สีเหลือง) ด้านบน

### ขั้นตอนที่ 3: ระบบติดตั้งฐานข้อมูลเริ่มต้น
1. ที่แถบเครื่องมือปรับแก้การทดสอบของหน้าต่าง Apps Script ให้เลือกฟังก์ชันแสดงชื่อชื่อว่า `initDatabase` แล้วกดปุ่ม คอนฟิก **"เรียกใช้" (Run)**
2. ระบบจะร้องขอการยืนยันสิทธิ์ความปลอดภัย ให้กดเลือก **"ตรวจสอบสิทธิ์สเปรดชีต"** และกดอนุมัติ (Allow) เพื่อให้โปรแกรมเข้าถึงและปรับปรุงชีตได้
3. *เมื่อพร้อมเสร็จสิ้น* โครงชีตสเกลฐานข้อมูลประวัติถังและสถิติเริ่มต้นของ ซีพีแรม (สุราษฎร์ธานี) จะถูกประมวลผลเพิ่มขึ้นมาบน Google Sheets เดิมของท่านด้วยความพร้อมสรรพ

### ขั้นตอนที่ 4: เผยแพร่เว็บแอป (New Deployment) เพื่อเชื่อมหน้าเว็บ
1. สังเกตที่ปุ่มสีฟ้าริมขวาบนชื่อว่า **"การให้บริการใช้งาน" (Deploy)** -> ให้กดแล้วเลือก **"การนำไปใช้งานใหม่" (New Deployment)**
2. กดสัญลักษน์ฟันเฟือง เลือกประเภทเป็น **"เว็บแอป" (Web App)**
3. ป้อนข้อความรายละเอียด และกำหนดสิทธิ์การใช้งานสำคัญดังนี้:
   * **ผู้ดำเนินการแอป (Execute as)**: เลือก *ฉัน (อีเมลสเปรดชีตของท่าน)*
   * **ผู้มีสิทธิ์เข้าถึงหน้าเว็บ (Who has access)**: เลือก *ทุกคน (Anyone)*
4. กดปุ่ม **"การให้บริการใช้งาน"**
5. คัดลอกพอยเตอร์ค่า **"URL ของเว็บแอป" (Web App URL)** ที่ปรากฏขึ้นมาล่าสุด เพื่อนำไปเข้าเปิดทดสอบการส่งรายงานและลงชื่อใช้งานระบบ

## 🍏 วิธีนำไปติดตั้งใช้งานบนระบบ iOS (Apple iPhone/iPad) เป็นแบบแอปพลิเคชันเสมือนจริง

ด้วยคุณลักษณะ Progressive Web App (PWA) หน้าตาและอินสแตนซ์ของหน้าเว็บแอปจะรับช่วงจัดการเหมือนการติดตั้งแอปจาก Store:

1. นำ **URL ของเว็บแอป** ที่ได้ทำการ Deploy มาส่งเปิดใช้งานผ่านโปรแกรมเว็บเบราว์เซอร์ **Safari** บนเครื่อง iPhone ของพนักงาน
2. สังเกตแถบไอคอนบาร์การควบคุมด้านล่าง (ปุ่มแชร์ที่มีลูกศรชี้ขึ้น) แล้วกดเปิด
3. เลื่อนหาเมนูชื่อว่า **"เพิ่มไปยังหน้าจอโฮม" (Add to Home Screen)**
4. ตั้งชื่อไอคอนแสดงที่หน้าหลักตัวอย่างเป็น `CPRAM Fire` และกด **"เพิ่ม" (Add)**
5. แฟ้มไอคอนแอปพลิเคชันจะมาปรากฏขึ้นบนหน้าหลักของโทรศัพท์พนักงานโดยอัติโนมัติ โดยเมื่อกดสแกนใช้งาน ระบบจะพริ้วตาส่วนประกอบแอนิเมชันให้เหมือนแอปแท้ แสดงความรู้สึกกะทัดรัด ไม่มีกรอบเบราว์เซอร์กวนใจ รองรับการทำงานในความเร็วของกล้องมือถือสูงมาก!
